### reset database
```bash
prisma migrate reset
```
### to create migration 
```bash
prisma migrate dev
```
### to run seed
```bash
prisma db seed
```
### to create nest resource
```bash
nest g resource
```
### to deploy a new migration 
```bash
prisma migrate deploy
``` 
### to install prisma cli
```bash
npm i -g prisma
``` 
### to install nest cli
```bash
npm i -g @nestjs/cli
``` 
### to start a nest project
```bash
npx nest new projectName
``` 
OR 
```bash
nest new projectName
``` 
### to add prisma 
```bash
npx prisma init
``` 
OR 
```bash
prisma init
```
